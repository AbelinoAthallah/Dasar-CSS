Point Style

CSS dapat mengatur style dari teks yang ada pada halaman web ang dibuat. Terdapat beberapa property yang dapat digunakan untuk mengatur style teks yaitu sebagai berikut.

  • text-align: digunakan untuk mengatur perataan teks,value yang dapat digunakan yaitu left, center, right.
  • color: digunakan untuk memberikan warna pda teks.
  • letter-spacing: mengatur jarak antar karakter.
  • word-spacing: mengatur jarak antar kata.
